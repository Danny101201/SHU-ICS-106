若你某天失去了雙眼，世界會變成什麼樣子呢？視識你帶領大家深入的認識視障者的生活全貌，也希望社會大眾能更加瞭解且看見視障朋友。

視障生活

1.視障如何使用手機?

如果說科技改變了我們的生活，那麼科技改變的是他們的命運。隨著科技的發展，視障人士已經可以使用手機訂外賣、打車、網購等，完成以前他們無法想像的事情。

Read more

2.AI科技視障擴視機

一台由人工智慧打造AI輔具眼鏡，解決視障人士生活所需，只要簡單手勢動作，就可以完成文字閱讀、人物辨識、鈔票辨識、語音報時等生活需求。

Read more

3.盲人科技產品 「盲人電腦」

盲人電腦與正常電腦差別在於，在電腦上加裝特殊閱讀軟體與給盲人使用的點字觸控板以及八鍵輸入鍵盤，只要將這三件輔助器具加上正常電腦也可以成為盲用電腦。

Read more

4.弱視福音視障輔具 「放大鏡」

目前國內所通行的弱視者用的望眼鏡，都是手持形的。國外有「鏡框式望眼鏡」方便閱讀和行動。手持式望眼鏡，適用於弱視生觀看遠距離的物體或文字，弱視者所用的望眼鏡是單筒，而且有放大效果。

Read more

5.視障教學點字輸入

盲人輸入結合點字，便於視障者使用盲人電腦，有效更快速的在網路上學習。

Read more

6.盲人手套突破新感官

一款由德國設計師jakob kilian所創造，透過偵測環境感知給予手套於不同位置發出震動給視障者回饋，讓障者就能判斷物體距離大小方位等相關訊息，手套還裝設力的反饋機制，讓感應裝置更能像觸覺般，讓視障朋友可以更清楚判別物品為何?
